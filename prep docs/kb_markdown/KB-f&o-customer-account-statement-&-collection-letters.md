**[Generate collection letters]{.underline}**

1\. Go to **Aged balances.**

2\. Select the customer.

3\. Click on \"New collection letters\".

![](media/image5.png){width="4.787211286089239in" height="1.9931681977252844in"}

4\. Make sure parameters are set correctly.

5\. Click OK.

![](media/image6.png){width="2.7176388888888887in" height="5.558245844269466in"}

**[Exclude specific Invoice from collection letters]{.underline}**

1\. Go to the customer account and open the **Transactions.**

![](media/image7.png){width="2.377665135608049in" height="1.67875656167979in"}

2\. Click on the voucher number you want to exclude from the collection letters.

![](media/image8.png){width="3.138513779527559in" height="2.123629702537183in"}

3\. Change to Edit mode so you can change the button **Collection letter** YES to NO.

![](media/image9.png){width="3.752749343832021in" height="3.713665791776028in"}
