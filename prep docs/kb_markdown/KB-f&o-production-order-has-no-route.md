When a production order is scheduled and there is no workplace assigned it is mainly because the production order has no route. When a production order has no route, item master data setup is incorrect.

# Procedure create a route version for the item

1.  Go the released item by clicking on the item number

![A screenshot of a computer AI-generated content may be incorrect.](media/image4.png){width="6.274178696412949in" height="2.013076334208224in"}

2.  Go to the tab engineer \> view \> Route

![A screenshot of a computer AI-generated content may be incorrect.](media/image5.png){width="6.152110673665792in" height="1.8989337270341207in"}

3.  If there is no route version, add a route version

![A screenshot of a computer AI-generated content may be incorrect.](media/image6.png){width="3.7897681539807526in" height="2.2946620734908136in"}

4.  Select the correct route and enter the site

![A screenshot of a computer AI-generated content may be incorrect.](media/image7.png){width="5.727830271216098in" height="2.3524343832020995in"}

Remark: a route has been created for each operation and runtime, select the correct one.

5.  Don't change anything else!

6.  Approve and activate the route version

![A screenshot of a computer AI-generated content may be incorrect.](media/image8.png){width="5.670509623797026in" height="1.6384536307961506in"}

# Procedure copy the route version to the production order

1.  Select the production order

2.  Reset the status to created

![A screenshot of a computer AI-generated content may be incorrect.](media/image9.png){width="6.268055555555556in" height="2.0625in"}

3.  Go to the production route

![A screenshot of a computer AI-generated content may be incorrect.](media/image10.png){width="6.2310061242344705in" height="2.5860126859142607in"}

4.  Click on Functions \> Copy

![A screenshot of a computer AI-generated content may be incorrect.](media/image11.png){width="5.709010279965004in" height="2.7393897637795277in"}

5.  Enter the item number from the production order

![A screenshot of a computer AI-generated content may be incorrect.](media/image12.png){width="2.6659919072615925in" height="4.426417322834646in"}

Remark: If everything is correct, the site and rout number will be filled in automatically.

6.  Click OK

7.  The route is copied to the production order

8.  Estimate the production order

9.  Reschedule the production order
