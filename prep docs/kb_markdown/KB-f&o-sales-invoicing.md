# KB — F&O Sales invoicing

**Systeem:** Pollet Group IT
**Bron:** Automatisch geconverteerd van `F&O Sales invoicing.docx`
**Conversiedatum:** 2026-04-24

---

1.  Navigate to **Make invoices.**

2.  It is possible to filter on **Pool**: local responsible per customer.

3.  **Select** the packing slips that you want to invoice. You can only invoice multiple packings slips together if they have the same **Invoicing method.**

4.  Via **Lines** check if all requirements are met:

    a.  Cost price is filled in.

    b.  Sales price is filled in. (you can see if an item is free)

    c.  Mark-up is ok or has been approved.

5.  Click on **Make invoices.**

6.  Make sure the printer settings are correct (Use print management) and click **OK.**

**[Remarks:]{.underline}**

- If you have to go to the sales order to make changes. You will have to refresh the Make invoices menu and re-select the lines.

- Items sold via the warehouse management flow or direct delivery are invoiced based on the packing slip.

- Items sold administratively can be invoiced directly from the sales order without a packing slip.

- When the **invoicing method** is **Per order** or **Per packing slip**, every packing slip needs to be invoiced separately.
