# KB — F&O External item descriptions

**Conversiedatum:** 2026-04-24

---

In case there are external item descriptions for an item, these will automatically appear on sales or purchase order lines.

For purchasing, we show the external description of the vendor from whom we are ordering unless there is a different (AX) MDC manufacturer (Default vendor) listed on the item, then this external description will be shown. You can always choose a different description in the **External** field in de line details.

Because AX is the leading source, you have to add new descriptions via AX and interface them to D365. The interface is triggered when you add a new description and when you click on Save.
