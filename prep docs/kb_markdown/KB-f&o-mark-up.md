- **Mark-up validation** in Sales orders when you generate the order confirmation!

> Mark-up approved ![Checkmark with solid fill](media/image3.svg){width="0.1730074365704287in" height="0.1730074365704287in"}

- If the mark-up is NOT OK it has to be approved! Otherwise, invoicing is NOT possible.

- For sales bundles, the mark-up check is on the bundle parent item.
