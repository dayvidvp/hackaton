A bundle (KITO) are components that are sold together. In an SO, you can recognize a bundle by the Icon In the column \'Isbundle\'. To explode the bundle In Its components there are 3 options: Click the bundle Icon, Via the button Bundle \--\> Explode bundle lines or by confirming the SO.

![A screenshot of a computer Description automatically generated](media/image2.png){width="6.268055555555556in" height="1.4305555555555556in"}

After explosion, the bundle Item Is Canceled and all the bundle components are shown. Internally, everything will now be done with the components while for external documents the bundle Itself Is still used.

![A screenshot of a computer Description automatically generated](media/image3.png){width="6.268055555555556in" height="1.15625in"}
