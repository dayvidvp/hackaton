# Create a new quotation

[Navigate to: Sales and marketing \> Sales quotations \> All quotations]{.underline}

1.  Click on **+New**.

2.  Select the **Customer account.**

3.  Set the correct **Delivery address** if necessary.

4.  Fill in the **Customer requisition** and **Customer reference.**

5.  Set the **Expiration date.**

6.  Select the **Requested ship date.**

7.  Click **OK.** The quotation is now created.

8.  Add the necessary lines:

    a.  Click on **+Add line** if you have the item number available.

    b.  Click on **+Product selection** if you need to search for the item.

    c.  Don\'t forget the **Configuration** in case in applies.

9.  For each line:

    a.  Fill in the **Quantity.**

    b.  Check/fill in/edit the **Unit price.**

    c.  Add any necessary **Discount.**

10. When the quotation is complete **Save** and navigate to **Quotation \> Generate**, click on **Send quotation.**

11. Make sure your **Printer setup** is correct and **Print quotation** is set to **Yes.**

12. Click **OK.**

> *Remark: At this point, there will be a \"Document validation\" which checks if all prices are filled in and the mark-up is ok or approved if necessary.*

# Follow up on an existing quotation

[Navigate to: Sales and marketing \> Sales quotations \> All quotations]{.underline}

1.  Find the specific quotation in the list. Use filters if necessary.

2.  Select the quotation by clicking in the first column.

There are 3 ways to follow up on a quotation: Confirm, Cancel, Lost quotation

3.  Choose your follow up and a new pop-up screen will appear.

4.  Select the reason for your follow up from the drop-down list.

In case of **Cancel/Lost quotation**, you are now ready.

In case of **Confirm**, the system will now propose to send a quotation confirmation. Continue without any printer settings. The system will create a sales order with the selected lines.

5.  You can navigate to: **General \> related information** and click on **Sales orders** to open it.

# Send attachments with quotations

**[An attachment linked to a released product is added automatically to the quotation.]{.underline}**

- Generate the quotation.

- If there are attachments linked to released products in the quotation, they will appear with the attachments of the email. (You also have a possibility to remove them)

- Send the email.

**[An attachment is added to the quotation.]{.underline}**

- In the quotation, click on the Raptor icon in the top right corner of your screen.

- Drag and drop the attachment in Raptor.

- Save.

- Generate the quotation, the attachment will appear in the email.

- Send the email.
